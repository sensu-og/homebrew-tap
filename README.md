# homebrew-tap

Homebrew formulas.

## General info

To install a formula:

```
brew update
brew tap sensu/tap
```

Then:

```
brew install sensu/tap/<formula-name>
```

## Formulas

#### [`sensu`](https://github.com/sensu/sensu)

```
brew update
brew tap sensu/tap
brew install sensu/tap/sensu
```
